# Kubernetes 기초

이 섹션에서는 Kubernetes의 기본 개념과 리소스를 실습합니다.

## 학습 목표

* Deployment의 기본 개념과 사용법 이해
* Service의 종류와 용도 파악
* ConfigMap과 Secret을 통한 설정 관리
* 배포 전략 (Blue-Green, Canary) 이해

## 실습 순서

1. [기본 Deployment](./basic-deployments) - Kubernetes Deployment의 기본 개념
2. [Service](./services) - 서비스 타입과 네트워킹
3. [ConfigMap](./configmaps) - 환경 설정 관리
4. [Secret](./secrets) - 민감 정보 관리
5. [Blue-Green 배포](./blue-green-deployments) - 무중단 배포 전략
6. [Canary 배포](./canary-deployments) - 점진적 배포 전략
